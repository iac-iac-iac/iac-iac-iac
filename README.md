## Избранные проекты

### [LeadManager_TGBot](https://github.com/iac-iac-iac/LeadManager_TGBot)

Telegram-бот для управления холодными лидами с интеграцией в Bitrix24 CRM. Помогает вести лиды, фиксировать статусы и ускорять работу с холодными контактами.

### [LeadGen_v5](https://github.com/iac-iac-iac/LeadGen_v5)

Десктопная система автоматизации лидогенерации для холодных продаж. Обрабатывает выгрузки из Яндекс.Карт и 2GIS, помогает структурировать базу и готовить данные к работе в CRM.

### [mita-marketing-agency](https://github.com/iac-iac-iac/mita-marketing-agency)

Сайт и платформа для М.И.Т.А. — маркетингового IT-агентства полного цикла. Стек: Next.js 14, TypeScript, SQLite CMS.
